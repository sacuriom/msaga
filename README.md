**A Native 3D Digital Twin and Memetic Approach for Smart City Infrastructure Optimization.**

This repository contains the source code, Digital Twin extraction tools, and 3D simulation assets associated with the research paper: *"Automated LoRaWAN Network Planning in Complex Urban Scenarios: A Native 3D Digital Twin and Memetic Approach"*, submitted to MDPI Sensors (2025).

## 🚀 Overview

The **Native 3D Memetic Spatially-Aware Genetic Algorithm (3D-M-SAGA)** represents a paradigm shift in Low-Power Wide-Area Network (LPWAN) planning. Moving beyond traditional 2D isotropic grids that systematically overestimate coverage, this framework generates a site-specific **3D Morphological Digital Twin**. It fuses vector-based OpenStreetMap (OSM) topologies with NASA SRTM 30m elevation data and an autonomous urban clutter classifier.

By pre-computing deterministic RF link budgets (accounting for ITU-R Knife-Edge diffraction and dielectric absorption), the algorithm optimizes Capital Expenditure (CAPEX) while guaranteeing mission-critical network health and fault tolerance ($k \ge 2$).

### 🌟 Key Features
* **Native 3D Digital Twin:** Real-time extraction of public road intersections (OSM) merged with topographic altitude data (NASA SRTM API).
* **Morphological Clutter Engine:** Autonomous classification of urban density (Dense Urban, Residential, Park/Periphery) for accurate path-loss exponent ($n$) assignment.
* **ITU-R Physical Constraints:** Integrates hardware specifications (Antenna Height, Tx Power, Gain) and Quality of Service (QoS) mapping based on LoRaWAN **Spreading Factors (SF7-SF12)**.
* **Vectorized Memetic Operator:** The "Smart Repair" algorithm uses heuristic attraction/repulsion to prune redundant CAPEX while filling physical coverage holes.
* **Academic Export Suite:** Automatically generates 600 DPI High-Resolution 3D PNGs, interactive MATLAB `.fig` files, detailed CSV topology logs, and comprehensive HTML deployment reports.

## 📂 Repository Contents

* `MDPI_Suite_MesoCellular_Academic.m`: The core MATLAB suite containing the Digital Twin builder, RF propagation engine, optimization algorithms, and 3D rendering module.
* `/Results_Meso_[City]_SF[X]_[Timestamp]/`: Autogenerated directories containing the simulation outputs:
  * `Academic_Deployment_Report.html`: Detailed QoS breakdown by Spreading Factor, K-Coverage metrics, and execution logs.
  * `DigitalTwin_[Algorithm].csv`: Raw spatial and RF metadata for every intersection.
  * `3D_Map_[Algorithm].png`: 600 DPI static renderings for paper publication.
  * `Native_Fig_[Algorithm].fig`: Interactive 3D MATLAB figures preserving DataTips.

## ⚙️ Installation & Usage

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/sacuriom/msaga.git](https://github.com/sacuriom/msaga.git)
    cd msaga
    ```

2.  **Run the Simulation Suite:**
    Open MATLAB, load the main script, and execute it:
    ```matlab
    MDPI_Suite_MesoCellular_Academic
    ```

3.  **Configure the Deployment:**
    * **Step 1:** Select a target city from the popup dialog (e.g., *Ambato, ECU*).
    * **Step 2:** A parameterization UI will appear. Input the desired **Spreading Factor (7 to 12)**, **Antenna Height (m)**, and **Target Coverage (%)**.
    * **Step 3:** The framework will automatically download the 3D topology, compute millions of physical RF links, run the algorithmic benchmark, and export the academic assets to a timestamped folder.

## 📊 Benchmarked Algorithms

The suite evaluates four distinct methodologies to highlight the Pareto-optimality of the proposed approach:
1.  **Greedy SCP:** A deterministic Set Covering Problem heuristic minimizing CAPEX but often resulting in fragile networks ($k=1$).
2.  **K-Medoids 3D:** A spatial clustering approach demonstrating the failure of pure geometric distribution when ignoring RF physical barriers.
3.  **Standard SAGA:** A classic genetic algorithm prone to epistatic variance and over-provisioning.
4.  **Proposed 3D-M-SAGA:** Our hybrid memetic algorithm balancing infrastructure cost with ideal network redundancy ($k \ge 2$).
